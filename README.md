# Wi-Fi Controlled Home Automation System using ESP32

## Overview
This project enables users to control electrical appliances remotely through a Wi-Fi network using an ESP32 and relay module.

## Features
- Wireless Appliance Control
- Web Dashboard
- Multiple Relay Support
- Low Cost Smart Home Solution

## Components
- ESP32
- 2-Channel Relay Module
- Wi-Fi Network
- Jumper Wires

## Working
The ESP32 hosts a web server. Users connect to the ESP32 IP address and use buttons on the web page to switch appliances ON and OFF.
